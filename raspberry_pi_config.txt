# 4
 - In main.py there will be 2 variables
   (1) position: Array for determining the movement of robot
        [1, 0, 0, 0] = Move forward
        [0, 1, 0, 0] = Move forward with caution
        [0, 0, 1, 0] = Move left
        [0, 0, 0, 1] = Move right

    (2) color_detected: Array for detected colors in the image
        - Length: 5
        - Indices: [blue, green, yellow, orange, red]
        - Eg. output: [1, 1, 0, 0, 0] (Blue and green detected)
